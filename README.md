# FCM Digital - Ruby Technical challenge

Ruby library that transforms user reservations into a easy-to-follow itinerary.

## Setup

- Ruby version.
- Bundle install.
- Tests

### Run the project

- `BASED=SVG bundle exec ruby main.rb input.txt`

### YARD documentation

This project has been documented using [YARD](https://yardoc.org/). To preview the documentation just run `bin/yard server` and open the given url.
